# Pakasir Proxy

Langkah Deploy:
1. Upload file index.js, package.json, .gitignore ke GitHub
2. Deploy ke Vercel
3. Tambahkan Environment Variable API_KEY di Vercel
   - Key: API_KEY
   - Value: API key Pakasir kamu
4. URL proxy akan tersedia dari Vercel, misal: https://pakasir-proxy.vercel.app
5. Gunakan URL proxy di HTML untuk generate QRIS
